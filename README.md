# CI/CD Pipeline Project ⚙️

A complete automated DevOps pipeline built with Jenkins, Docker, and GitHub.

## 🚀 Pipeline Flow
GitHub → Jenkins → Install → Test → Docker Build → Deploy

## 🛠️ Tech Stack
- **Node.js** — Backend application
- **Jenkins** — CI/CD automation
- **Docker** — Containerization
- **GitHub** — Source control

## ⚡ Features
- Automatic build on code push
- Automated testing
- Docker image creation
- Auto deployment on port 3000

## 🏃 How to Run Locally
```bash
npm install
node app.js
```
Visit: http://localhost:3000
